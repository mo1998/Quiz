# Quiz App
Simple online quiz app using html js css php sql.

## How to use?
1. Download this repo.
2. Paste it in xampp/htdocs/ .
3. Create new database naming "exam".
4. Import "exam.sql" file.
5. Open a browser and go to URL"http://localhost/Quiz".
